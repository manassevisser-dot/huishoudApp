## 🔧 Micro‑patch (exacte regels, minimaal)

**Bestand:** `App.tsx` (root), in de functie **`navigateNext`**  
Zoek het blok waar je bepaalt dat de next stap **C4** is (jij hebt al `c4Index = useMemo(...)` en logic in `handleSignin/handleSignup`/`navigateNext`).  
**Vóór** het `setCurrentPageIndex(...)` naar `c4Index`, **voeg**:

```ts
// Just-in-time alignment wanneer de volgende pagina C4 is
if (c4Index >= 0 && currentPageIndex === 0 /* van C1 naar C4 */) {
  const a = Number(state.C1?.aantalMensen ?? 0);
  const bRaw = Number(state.C1?.aantalVolwassen ?? 0);
  // B-regel expliciet: min 1 volwassene, nooit meer dan mensen
  const b = Math.min(a, Math.max(1, bRaw));

  // Dispatch ALIGN met actuele counters uit state vlak voor we C4 binnenkomen
  dispatch({
    type: 'ALIGN_HOUSEHOLD_MEMBERS',
    payload: { aantalMensen: a, aantalVolwassen: b },
  });
}
```

> **Waarom hier?**  
> Dit garandeert dat **net voor** C4, `leden[]` in sync is met **actuele A/B**.  
> Het is tijdstip‑correct en **deterministisch** (geen reliance op hydrate‑timing).
