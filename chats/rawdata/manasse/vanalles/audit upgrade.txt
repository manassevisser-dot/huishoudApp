# 🔍 Phoenix Scripts Analyse & Aanbevelingen

Ik heb je scripts doorgenomen. Je hebt een solide basis gebouwd, maar er zijn enkele verbeterpunten voor een cleane werkdag-afsluiting:

## 🚨 Kritieke Issues

### 1. **Inconsistente Bridge Implementatie**
```bash
# phoenix_dedup.sh (document 1)
# Importeer de bridge
# <- Deze regel doet NIETS
```
De bridge wordt aangekondigd maar nooit geïmporteerd. Dit moet zijn:
```bash
source "${PROJECT_ROOT}/scripts/utils/log_bridge.sh"
```

### 2. **Ontbrekende log_bridge.sh**
Meerdere scripts refereren naar `log_bridge.sh` maar dit bestand ontbreekt in je documentatie. Dit is je **Single Point of Failure**.

### 3. **Dubbele Phoenix Bestanden**
Je hebt `phoenix (1).txt` - waarschijnlijk een duplicate. Dit creëert verwarring over welke versie actueel is.

## ⚠️ Belangrijke Verbeteringen

### 4. **Error Handling in Bash**
```bash
# cleanup_and_commit.sh
npm run -s lint  >/dev/null 2>&1 || log_val "warn" "CHECK_ISSUE" "lint"
```
Dit maskeert alle errors. Beter:
```bash
if ! npm run -s lint 2>&1 | tee lint.log; then
    log_val "warn" "CHECK_ISSUE" "lint"
fi
```

### 5. **Race Condition in Lock Mechanisme**
```bash
# phoenix-check.sh
if mkdir "$LOCKDIR" 2>/dev/null; then
  # Success path
else
  # Check stale lock
fi
```
Dit is niet atomisch genoeg. Gebruik `flock` voor betrouwbare locking:
```bash
exec 200>/var/lock/phoenix-audit.lock
flock -n 200 || { log_err "LOCK_ACTIVE"; exit 1; }
```

### 6. **Hardcoded Paths**
```javascript
// index.js
const ROOT = path.resolve(__dirname, '../../');
```
Dit breekt als scripts vanuit andere locaties draaien. Gebruik:
```javascript
const ROOT = process.env.PROJECT_ROOT || path.resolve(__dirname, '../../');
```

### 7. **Missing Rollback Functionaliteit**
`migrate-field-id.sh` heeft backup maar geen rollback command. Voeg toe:
```bash
cmd_rollback() {
    local latest_backup=$(ls -t backups/*.tar.gz | head -1)
    tar -xzf "$latest_backup" -C "$PROJECT_ROOT"
}
```

## 🎯 Ontbrekende Functionaliteit

### 8. **Pre-commit Hook**
Maak `.git/hooks/pre-commit`:
```bash
#!/usr/bin/env bash
./phoenix audit || {
    echo "❌ Audit failed. Commit blocked."
    exit 1
}
```

### 9. **CI/CD Integration Check**
Voeg `ci-check.sh` toe:
```bash
#!/usr/bin/env bash
# Detecteer CI omgeving
if [[ -n "${CI:-}" ]]; then
    export NO_COLOR=1
    export QUIET=true
fi
./phoenix all --dry-run
```

### 10. **Dependency Verification**
Scripts assumeren tools bestaan. Voeg centraal toe in `phoenix`:
```bash
check_dependencies() {
    local deps=(node npm bash grep awk sed)
    for cmd in "${deps[@]}"; do
        command -v "$cmd" >/dev/null || {
            log_err "Missing dependency: $cmd"
            exit 1
        }
    done
}
```

## 📊 Code Quality Issues

### 11. **Magic Numbers**
```javascript
// index.js
const MAX_FILE_SIZE = 50000; // 50KB limiet
```
Verplaats naar config:
```javascript
// config.js
module.exports = {
    limits: {
        maxFileSize: 50 * 1024, // 50KB
        maxReports: 5,
        lockTimeout: 1800 // 30 min
    }
}
```

### 12. **Inconsistente Exit Codes**
Scripts gebruiken random exit codes. Standaardiseer:
```bash
# constants.sh
EXIT_SUCCESS=0
EXIT_LOCK_HELD=10
EXIT_CONFIG_ERROR=20
EXIT_VALIDATION_ERROR=30
```

### 13. **No Dry-run in Critical Operations**
```bash
# cleanup_and_commit.sh - Git operations hebben geen dry-run mode
git commit -m "$commit_msg"
```

## 🔒 Security Concerns

### 22. **Unsafe `eval` Potential**
```bash
log_node() {
    node -e "const l=require('${PROJECT_ROOT}/scripts/utils/logger'); $1"
}
```
Dit kan command injection veroorzaken. Sanitize input.

### 23. **Unquoted Variables**
Veel plaatsen waar `"$var"` moet zijn in plaats van `$var`

## 🎓 Best Practices

### ✅ DO's

1. **Use Descriptive Names**
   ```typescript
   ✅ it('moet duplicaat IDs uniek maken', ...)
   ❌ it('test 1', ...)
   ```

2. **Test One Thing**
   ```typescript
   ✅ it('moet null namen afhandelen', ...)
   ✅ it('moet lege namen afhandelen', ...)
   ❌ it('moet alle edge cases afhandelen', ...)
   ```

3. **Arrange-Act-Assert Pattern**
   ```typescript
   it('must transform correctly', async () => {
     // Arrange
     const input = createTestData();
     
     // Act
     const result = await migrate(input);
     
     // Assert
     expect(result).toBeDefined();
   });
   ```

4. **Use Test Helpers**
   ```typescript
   ✅ const data = testDataGenerators.createLedenBatch(10);
   ❌ const data = { leden: [...] }; // Handmatig
   ```

5. **Assert Strongly**
   ```typescript
   ✅ expect(members).toHaveLength(2);
   ✅ expect(members[0].naam).toBe('Jan');
   ❌ expect(members.length).toBeGreaterThan(0);
   ```

---

### ❌ DON'Ts

1. **Don't Test Implementation Details**
   ```typescript
   ❌ expect(internalFunction).toHaveBeenCalled();
   ✅ expect(result.data).toBeDefined();
   ```

2. **Don't Use Magic Numbers**
   ```typescript
   ❌ expect(members).toHaveLength(42);
   ✅ const EXPECTED_COUNT = 42;
      expect(members).toHaveLength(EXPECTED_COUNT);
   ```

3. **Don't Skip Cleanup**
   ```typescript
   ✅ afterEach(() => jest.clearAllMocks());
   ```

4. **Don't Make Tests Dependent**
   ```typescript
   ❌ // Test 2 depends on Test 1
   ✅ // Each test is independent
   ```

5. **Don't Ignore Async**
   ```typescript
   ❌ it('test', () => { migrate(data); });
   ✅ it('test', async () => { await migrate(data); });
   ```

#!/usr/bin/env bash
# install-phoenix.sh - Install or upgrade Phoenix scripts
set -euo pipefail

# Colors
GREEN='\033[0;32m'
YELLOW='\033[0;33m'
RED='\033[0;31m'
BLUE='\033[0;34m'
BOLD='\033[1m'
NC='\033[0m'

echo -e "${BLUE}${BOLD}"
cat <<'BANNER'
    ____  __                _     
   / __ \/ /_  ____  ___  (_)  __
  / /_/ / __ \/ __ \/ _ \/ / |/_/
 / ____/ / / / /_/ /  __/ />  <  
/_/   /_/ /_/\____/\___/_/_/|_|  
                                 
Phoenix Script Suite Installer v3.3
BANNER
echo -e "${NC}"

# === Configuration ===
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
PROJECT_ROOT="$(cd "$SCRIPT_DIR/../.." 2>/dev/null && pwd || pwd)"
BACKUP_DIR="${PROJECT_ROOT}/backups/phoenix-$(date +%Y%m%d_%H%M%S)"

# === Parse Arguments ===
SKIP_BACKUP=false
FORCE=false

for arg in "$@"; do
    case $arg in
        --skip-backup) SKIP_BACKUP=true ;;
        --force) FORCE=true ;;
        -h|--help)
            cat <<HELP
Usage: ./install-phoenix.sh [options]

Options:
  --skip-backup    Skip backing up existing files
  --force          Overwrite existing files without prompting
  -h, --help       Show this help

HELP
            exit 0
            ;;
    esac
done

# === Pre-flight Checks ===
echo -e "${BLUE}📋 Pre-flight Checks${NC}"
echo ""

# Check if we're in a git repo
if ! git rev-parse --git-dir > /dev/null 2>&1; then
    echo -e "${YELLOW}⚠️  Not in a git repository${NC}"
    read -p "Continue anyway? (y/N) " -n 1 -r
    echo
    [[ ! $REPLY =~ ^[Yy]$ ]] && exit 0
fi

# Check for required tools
echo "Checking dependencies..."
missing=()
for tool in node npm bash grep awk sed; do
    if ! command -v "$tool" >/dev/null 2>&1; then
        missing+=("$tool")
    fi
done

if [[ ${#missing[@]} -gt 0 ]]; then
    echo -e "${RED}❌ Missing required tools: ${missing[*]}${NC}"
    echo "Install them and try again."
    exit 1
fi
echo -e "${GREEN}✓ All dependencies present${NC}"

# Check Node.js version
node_version=$(node --version | sed 's/v//')
required_major=18
current_major=$(echo "$node_version" | cut -d. -f1)

if [[ $current_major -lt $required_major ]]; then
    echo -e "${YELLOW}⚠️  Node.js $node_version detected (v$required_major+ recommended)${NC}"
fi

# === Backup Existing Files ===
if [[ "$SKIP_BACKUP" == "false" ]]; then
    echo ""
    echo -e "${BLUE}💾 Creating Backup${NC}"
    
    mkdir -p "$BACKUP_DIR"
    
    # Backup existing Phoenix files
    for item in phoenix scripts/.phoenix scripts/maintenance scripts/utils/log_bridge.sh; do
        if [[ -e "$PROJECT_ROOT/$item" ]]; then
            cp -r "$PROJECT_ROOT/$item" "$BACKUP_DIR/" 2>/dev/null || true
        fi
    done
    
    if [[ -d "$BACKUP_DIR" ]] && [[ -n "$(ls -A "$BACKUP_DIR")" ]]; then
        echo -e "${GREEN}✓ Backup created: $BACKUP_DIR${NC}"
    else
        echo "  (No existing files to backup)"
    fi
fi

# === Create Directory Structure ===
echo ""
echo -e "${BLUE}📁 Creating Directory Structure${NC}"

directories=(
    "scripts/maintenance"
    "scripts/sync-aliases"
    "scripts/utils"
    "scripts/utils/constants"
    "scripts/git-hooks"
    ".phoenix"
    "backups"
    "reports"
)

for dir in "${directories[@]}"; do
    mkdir -p "$PROJECT_ROOT/$dir"
    echo "  ✓ $dir"
done

# === Install Core Files ===
echo ""
echo -e "${BLUE}📦 Installing Phoenix Files${NC}"

# Check if files should be overwritten
install_file() {
    local src="$1"
    local dest="$2"
    
    if [[ -f "$dest" ]] && [[ "$FORCE" == "false" ]]; then
        echo -e "${YELLOW}  File exists: $dest${NC}"
        read -p "  Overwrite? (y/N) " -n 1 -r
        echo
        [[ ! $REPLY =~ ^[Yy]$ ]] && return
    fi
    
    cp "$src" "$dest"
    chmod +x "$dest" 2>/dev/null || true
    echo -e "${GREEN}  ✓ Installed: $(basename "$dest")${NC}"
}

# Note: In a real scenario, these would be copied from the artifacts
# For now, we'll create placeholder detection
echo "  (Checking for files to install...)"
echo "  Manual step required: Copy artifact files to their locations"

# === Install Git Hooks ===
echo ""
echo -e "${BLUE}🪝 Git Hooks${NC}"

if [[ -f "$PROJECT_ROOT/.git/hooks/pre-commit" ]] && [[ "$FORCE" == "false" ]]; then
    echo -e "${YELLOW}  Pre-commit hook already exists${NC}"
    read -p "  Overwrite? (y/N) " -n 1 -r

#!/usr/bin/env node
const fs = require('fs');
const path = require('path');
const transports = require('./transports');
const sync = require('./constants/sync');
const audit = require('./constants/audit');
const generic = require('./constants/generic');

// === Exit Codes ===
const EXIT_CODES = {
    SUCCESS: 0,
    LOCK_HELD: 10,
    CONFIG_ERROR: 20,
    VALIDATION_ERROR: 30,
    DEPENDENCY_ERROR: 40,
    UNKNOWN_ERROR: 99,
};

// === Terminal Link Helper ===
const terminalLink = (text, filePath) => {
    const OSC = '\u001B]8;;';
    const ST = '\u001B\\';
    const url = `file://${path.resolve(filePath)}`;
    return `${OSC}${url}${ST}${text}${OSC}${ST}`;
};

// === Logger Object ===
const logger = {
    // Constants
    TEXT: { ...generic, ...sync, ...audit },
    EXIT_CODES,
    
    // Flags
    isVerbose: process.argv.includes('--verbose') || process.argv.includes('-v'),
    isDryRun: process.argv.includes('--dry-run'),
    isQuiet: process.argv.includes('--quiet'),
    isCI: !!process.env.CI,
    
    // Timer
    _startTime: null,
    _timers: new Map(), // Support multiple named timers
    
    // === Timer Methods ===
    startTimer: (name = 'default') => {
        logger._timers.set(name, Date.now());
        if (name === 'default') {
            logger._startTime = Date.now();
        }
    },
    
    stopTimer: (name = 'default') => {
        const start = logger._timers.get(name) || logger._startTime;
        if (!start) {
            logger.warn('No timer started');
            return 0;
        }
        
        const duration = ((Date.now() - start) / 1000).toFixed(2);
        
        if (!logger.isQuiet) {
            transports.writeStream('info', "\x1b[2m⏱️  Duration: " + duration + "s\x1b[0m");
        }
        
        logger._timers.delete(name);
        if (name === 'default') {
            logger._startTime = null;
        }
        
        return parseFloat(duration);
    },
    
    getElapsed: (name = 'default') => {
        const start = logger._timers.get(name) || logger._startTime;
        if (!start) return 0;
        return ((Date.now() - start) / 1000).toFixed(2);
    },
    
    // === Core Logging Methods ===
    info: (key) => {
        if (logger.isQuiet) return;
        transports.info(logger._resolveText(key));
    },
    
    ok: (key) => {
        if (logger.isQuiet) return;
        transports.ok(logger._resolveText(key));
    },
    
    warn: (key) => {
        transports.warn(logger._resolveText(key));
    },
    
    error: (key) => {
        transports.error(logger._resolveText(key));
    },
    
    verbose: (msg) => {
        if (logger.isVerbose && !logger.isQuiet) {
            transports.writeStream('verbose', "\x1b[2m🔍 " + msg + "\x1b[0m");
        }
    },
    
    debug: (msg) => {
        if (process.env.DEBUG) {
            transports.writeStream('debug', "\x1b[90m🐛 " + msg + "\x1b[0m");
        }
    },
    
    // === Report with Clickable Link ===
    report: (reportPath) => {
        if (logger.isQuiet) return;
        
        const link = terminalLink(reportPath, reportPath);
        const message = logger.TEXT.REPORTS_LOCATION
            ? logger.TEXT.REPORTS_LOCATION(link)
            : `📄 See reports in: ${link}`;
        
        transports.ok(message);
    },
    
    // === Fatal Error (with exit) ===
    die: (msgKey, exitCode = EXIT_CODES.UNKNOWN_ERROR) => {
        const msg = logger._resolveText(msgKey);
        const fatal = logger.TEXT.FATAL ? logger.TEXT.FATAL(msg) : msg;
        transports.die(fatal, exitCode);
    },
    
    // === Progress Indicator ===
    progress: (current, total, label = '') => {
        if (logger.isQuiet) return;
        
        const percent = Math.round((current / total) * 100);
        const bar = '█'.repeat(Math.floor(percent / 2)) + '░'.repeat(50 - Math.floor(percent / 2));
        
        process.stdout.write(`\r${label} [${bar}] ${percent}% (${current}/${total})`);
        
        if (current === total) {
            process.stdout.write('\n');
        }
    },
    
    // === Spinner ===
    spinner: (message) => {
        if (logger.isQuiet) return { stop: () => {} };
        
        const frames = ['⠋', '⠙', '⠹', '⠸', '⠼', '⠴', '⠦', '⠧', '⠇', '⠏'];
        let i = 0;
        
        const interval = setInterval(() => {
            process.stdout.write(`\r${frames[i]} ${message}`);
            i = (i + 1) % frames.length;
        }, 80);
        
        return {
            stop: (finalMessage = '') => {
                clearInterval(interval);
                process.stdout.write(`\r${finalMessage}\n`);
            }
        };
    },
    
    // === Structured Logging (JSON) ===
    json: (data) => {
        if (logger.isCI || process.env.JSON_LOGS) {
            console.log(JSON.stringify({
                timestamp: new Date().toISOString(),
                level: 'info',
                ...data
            }));
        }
    },
    
    // === Group/Collapse (for CI) ===
    group: (name) => {
        if (logger.isCI) {
            if (process.env.GITHUB_ACTIONS) {
                console.log(`::group::${name}`);
            } else {
                console.log(`\n=== ${name} ===`);
            }
        } else {
            console.log(`\n📦 ${name}`);
        }
    },
    
    groupEnd: () => {
        if (logger.isCI && process.env.GITHUB_ACTIONS) {
            console.log('::endgroup::');
        }
    },
    
    // === Helper: Resolve Text with Functions ===
    _resolveText: (key) => {
        const text = logger.TEXT[key];
        if (!text) return key;
        return typeof text === 'function' ? text() : text;
    },
    
    // === Config Getter ===
    getConfig: () => {
        return {
            verbose: logger.isVerbose,
            dryRun: logger.isDryRun,
            quiet: logger.isQuiet,
            ci: logger.isCI,
        };
    },
};

// === Export ===
module.exports = logger;

// === CLI Test Mode ===
if (require.main === module) {
    console.log('🧪 Logger Test Mode\n');
    
    logger.startTimer();
    logger.info('AUDIT_START');
    logger.ok('DEDUP_OK');
    logger.warn('LOCK_STALE');
    logger.error('LOCK_ACTIVE');
    logger.verbose('This is verbose output');
    
    logger.group('Test Group');
    logger.info('Inside group');
    logger.groupEnd();
    
    const spinner = logger.spinner('Processing...');
    setTimeout(() => {
        spinner.stop('✅ Done!');
        logger.stopTimer();
    }, 2000);
}

#!/usr/bin/env bash
# log_bridge.sh - Phoenix Logging Bridge v3.0
# Centralized bridge between Bash and Node.js logger
set -euo pipefail

# === Project Root Detection ===
if [[ -z "${PROJECT_ROOT:-}" ]]; then
    PROJECT_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/../.." && pwd)"
    export PROJECT_ROOT
fi

# === Logger Path ===
LOGGER_PATH="${PROJECT_ROOT}/scripts/utils/logger.js"

# === Color Fallbacks (if logger unavailable) ===
if [[ -t 1 ]] && [[ -z "${NO_COLOR:-}" ]]; then
    RED='\033[0;31m'
    GREEN='\033[0;32m'
    YELLOW='\033[0;33m'
    BLUE='\033[0;34m'
    DIM='\033[2m'
    BOLD='\033[1m'
    NC='\033[0m'
else
    RED=''; GREEN=''; YELLOW=''; BLUE=''; DIM=''; BOLD=''; NC=''
fi

# === Core Bridge Function ===
log_node() {
    local level="$1"
    local key="$2"
    shift 2
    local args=("$@")
    
    # Sanitize inputs to prevent injection
    key=$(printf '%s' "$key" | sed "s/'/\\\'/g")
    
    if [[ -f "$LOGGER_PATH" ]]; then
        # Build Node command safely
        local node_cmd="const l=require('${LOGGER_PATH}');"
        
        if [[ ${#args[@]} -gt 0 ]]; then
            # Dynamic key with arguments
            local sanitized_args=()
            for arg in "${args[@]}"; do
                sanitized_args+=("$(printf '%s' "$arg" | sed "s/'/\\\'/g")")
            done
            node_cmd+="l.${level}(l.TEXT['${key}'](${sanitized_args[*]}));"
        else
            # Static key
            node_cmd+="l.${level}('${key}');"
        fi
        
        node -e "$node_cmd" 2>/dev/null || {
            # Fallback to direct echo if node fails
            _log_fallback "$level" "$key" "${args[@]}"
        }
    else
        _log_fallback "$level" "$key" "${args[@]}"
    fi
}

# === Fallback Logger (no Node.js) ===
_log_fallback() {
    local level="$1"
    local msg="$2"
    shift 2
    local args=("$@")
    
    local full_msg="$msg"
    if [[ ${#args[@]} -gt 0 ]]; then
        full_msg="$msg: ${args[*]}"
    fi
    
    case "$level" in
        info)  echo -e "${BLUE}ℹ️  ${full_msg}${NC}" ;;
        ok)    echo -e "${GREEN}✅ ${full_msg}${NC}" ;;
        warn)  echo -e "${YELLOW}⚠️  ${full_msg}${NC}" ;;
        error) echo -e "${RED}❌ ${full_msg}${NC}" >&2 ;;
        *)     echo "$full_msg" ;;
    esac
}

# === Public API ===
log_info() {
    log_node "info" "$1"
}

log_ok() {
    log_node "ok" "$1"
}

log_warn() {
    log_node "warn" "$1"
}

log_err() {
    log_node "error" "$1"
}

# log_val: For dynamic messages with 1 argument
log_val() {
    local level="$1"
    local key="$2"
    local arg="$3"
    log_node "$level" "$key" "'$arg'"
}

# log_val2: For dynamic messages with 2 arguments
log_val2() {
    local level="$1"
    local key="$2"
    local arg1="$3"
    local arg2="$4"
    log_node "$level" "$key" "'$arg1'" "'$arg2'"
}

# log_val3: For dynamic messages with 3 arguments
log_val3() {
    local level="$1"
    local key="$2"
    local arg1="$3"
    local arg2="$4"
    local arg3="$5"
    log_node "$level" "$key" "'$arg1'" "'$arg2'" "'$arg3'"
}

# === Export Functions ===
export -f log_node log_info log_ok log_warn log_err log_val log_val2 log_val3 _log_fallback

# === Validation ===
if [[ "${BASH_SOURCE[0]}" == "${0}" ]]; then
    # Script is being run directly (test mode)
    echo "🧪 Testing log_bridge.sh..."
    log_info "AUDIT_START"
    log_ok "DEDUP_OK"
    log_warn "LOCK_STALE"
    log_err "LOCK_ACTIVE"
    echo "✅ Bridge test complete"
fi
