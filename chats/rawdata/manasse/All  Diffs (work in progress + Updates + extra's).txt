  * Frames: AdultOptions & ChildOptions
  * For each adult category: create dynamic controls with DYN_PREFIX
  * Show/hide child frame based on total >1
  * Add shape buttons using AddShapeAsImage from Referenties
- Event Handlers:
  * txtTotalPersons_Change → recalc visibility
  * AdultCount_Change, ChildCount_Change → recalc totals + enable/disable Next button

  * Use IsNumeric() before Val()
