
CHECKLIST VOOR CONTROLES:
=========================
1. Controleer DYN_PREFIX:
   - Alle referenties moeten zijn: FormLayout.DYN_PREFIX
   - Zoek naar: DYN_PREFIX zonder prefix, FormEvents.DYN_PREFIX, FormData.DYN_PREFIX
   - Vervang door: FormLayout.DYN_PREFIX

2. Enum gebruik:
   - Zoek naar Utils.dbDate, Utils.dbYear, etc.
   - Vervang door: dbDate, dbYear, dbWeek, dbAmount, dbCategory (Enum DBCols)

3. Foutafhandeling:
   - Zorg voor On Error GoTo ErrHandler
   - Voeg Exit Sub vóór ErrHandler
   - Gebruik Utils.LogError4 voor logging

4. Shapes voor achtergrond:
   - Controleer dat shape op blad "Referenties" heet: bg_frmInvoer_landscape
   - In FormBackground.GetBackgroundPath: Set shp = wsRef.Shapes("bg_frmInvoer_landscape")

5. Reporting module:
   - Verwijder Private Const WEEK_SUMMARY_COL en MONTH_SUMMARY_COL
   - Gebruik Enum SummaryCols: SummaryCols.colWeekSummaryStart, SummaryCols.colMonthSummaryStart

6. UserForm dynamische opbouw:
   - BuildDynamicFrmInvoer moet Application.ScreenUpdating = False/True gebruiken
   - HookEventsDynamic koppelt knoppen en comboboxen
